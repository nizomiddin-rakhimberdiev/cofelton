# Кофе с собой — Landing

Franshiza kofexonalari uchun landing sahifa. React + Vite.

## Xususiyatlar

- 🇷🇺🇺🇿 **2 til**: Rus va O'zbek
- 📱 **Responsive** dizayn
- 📊 **Daromad kalkulyatori** (rubl / so'm)
- 📤 **Telegram** orqali formalar yuboriladi
- 🎨 **Zamonaviy UI/UX**

## O'rnatish

```bash
npm install
```

## Telegram sozlash

1. `.env.example` ni `.env` ga nusxalang
2. [@BotFather](https://t.me/BotFather) da `/newbot` — bot yarating
3. [@userinfobot](https://t.me/userinfobot) ga `/start` — Chat ID oling
4. `.env` da qiymatlarni kiriting:

```
VITE_TELEGRAM_BOT_TOKEN=123456:ABC...
VITE_TELEGRAM_CHAT_ID=123456789
```

## Ishga tushirish

```bash
npm run dev
```

Build: `npm run build`
